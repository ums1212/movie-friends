## 04. 보완점 및 향후 계획

- 사용자 입력폼 유효성 체크 적용 필요

- 한정된 기한에 구현에만 몰두하다보니 Usecase 활용, 함수 분리 등 클린 코드, 클린 아키텍처 지향적인 리팩토링이 필요

- 미완성 기능 모두 구현하기

- 가능하다면 인원을 모집하여 팀 프로젝트로 진행

- 구글 플레이스토어 배포 계획

- 실제 배포가 이루어지면 영화 친구 만들어보기
